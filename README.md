# GB Studio Plugins

This repository contains a collection of plugins for [GB Studio](https://www.gbstudio.dev/), developed to extend the functionality of your Game Boy games.

## Available Plugins

### Multi Language Dialogue
**Location:** `/Multi_Languaje_Dialogue`

A plugin designed to simplify the implementation of multi-language dialogue systems in GB Studio.

- **Type:** Events Plugin
- **Author:** Tedezed
- **GB Studio Version:** >= 4.2.0
- **License:** MIT

## Installation

1. Download this repository or the specific plugin folder.
2. Place the plugin folder inside your GB Studio project's `plugins` directory (e.g., `your-project/plugins/Multi_Languaje_Dialogue`).
3. Reload your project in GB Studio.
